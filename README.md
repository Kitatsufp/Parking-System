# Parking-System
This final project for the Embedded Systems Design course involves designing a smart parking system that manages vehicle entry and exit and uses a servo motor as a barrier.
This is my second MCU-based system design. Unlike my first project, I implemented the entire system without using modules and programmed the MCU using register-level programming.
The firmware operated efficiently with no software errors. However, hardware instability occurred due to voltage drops. The issue was caused by directly using the MCU power supply to drive other devices. After identifying this problem, I improved the power design.
Due to the significant amount of time spent debugging and fixing hardware issues, the user interface was not fully developed. The system was implemented using a perforated prototyping board, which was not the most suitable choice for this project. A custom PCB, either handmade or professionally fabricated, would have provided better stability and reliability. This was a design decision based on my previous experience, where I commonly used prototyping boards in earlier projects.
